Variables and Datatypes

Variables and Datatypes

Variables are nothing but reserved memory locations to store values. This means that when you create a variable you reserve some space in memory.

A variable is created the moment you first assign a value to it.

number = 7
myName = "John"

Note : Python allows you to name variables to your liking, as long as the names follow these rules: Variable names may contain letters, digits (0-9) or the underscore character “_” . Variable names must begin with a letter from A-Z or the underscore “_” character. Either lowercase or uppercase letters are acceptable.

Acceptable variable names :

var = "neuraliqa"
v_a_r = "neuraliqa"
_var = "neuraliqa"
vAr = "neuraliqa"
VAR = "neuraliqa"
var2 = "neuraliqa"

Unacceptable variable names :

2var = "neuraliqa"
v-a-r = "neuraliqa"
v a r = "neuraliqa"

In Python, we can assign multiple variables in one line.

x = y = z = 5
x, y, z = "Orange", "Banana", "Apple"

String

Strings in Python are identified as a contiguous set of characters represented in the quotation marks. Example :

myStr = "neuraliqa"
print(myStr)

>>> neuraliqa

Some properties :

str = 'Hello World!'

print (str)          # Prints complete string
print (str[0])       # Prints first character of the string
print (str[2:5])     # Prints characters starting from 3rd to 5th
print (str[2:])      # Prints string starting from 3rd character
print (str * 2)      # Prints string two times
print (str + "SOMETHING") # Prints concatenated string
