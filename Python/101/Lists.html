Lists

You can think of a data structure as a way of organizing and storing data such that we can access and modify it efficiently.

There are four collection data types in the Python programming language:

  • List is a collection which is ordered and changeable. Allows duplicate members.
  • Tuple is a collection which is ordered and unchangeable. Allows duplicate members.
  • Set is a collection which is unordered, unchangeable, and unindexed. No duplicate members.
  • Dictionary is a collection which is ordered and changeable. No duplicate members.

Lists

A list is a collection of ordered data.

list1=[1,4,"Gitam",6,"college"]
list2=[]  # creates an empty list
list3=list((1,2,3))

print(list1)

>>> [1, 4, 'Neuraliqa', 6, True]

To access a list item :

mylist= ["apple", "banana", "cherry", "orange", "melon"]

print(mylist[1])
print(mylist[-1])
print(mylist[1:3])
print(mylist[:3])
print(mylist[2:])

if "apple" in mylist:
  print("Yes, 'apple' is in the fruits list")

>>> banana #Note: The first item has index 0.
>>> melon #-1 : last item
>>> ['banana', 'cherry'] #This will return the items from position 2 to 5.
>>> ['apple', 'banana', 'cherry']  #This example returns the items from the beginning to, but NOT including, "orange":
>>> ['cherry', 'orange', 'melon']
>>> Yes, 'apple' is in the fruits list

To change a list item :

mylist= ["apple", "banana", "cherry"]
mylist[1] = "blackcurrant"
print(mylist)

To insert an item :

mylist= ["apple", "banana", "cherry"]
mylist.insert(2, "watermelon")
#or
mylist.append("orange")
print(mylist)

To remove an item :

mylist = ["apple", "banana", "cherry"]
mylist.remove("banana")
mylist.pop(1) #Remove the second item
mylist.pop() #Remove the last item
del mylist[0] #Remove the first item
del mylist #Delete the entire list
thislist.clear()

Sort the list :

mylist = ["orange", "mango", "kiwi", "pineapple", "banana"]
mylist.sort()
mylist.sort(reverse = True) #reversed

Getting the length of a list :

mylist= ["apple", "banana", "cherry"]
print(len(mylist))

Copy a list :

mylist= ["apple", "banana", "cherry"]
mylist2 = mylist.copy()
#or
mylist3 = list(mylist)

Add two lists :

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list3 = list1 + list2
