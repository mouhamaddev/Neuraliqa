Basic Operations

Operators are symbols which tells the interpreter to do a specific operation such as arithmetic, comparison, logical, and so on.

Python has 7 types of Operators :

  • Arithmetic Operator
  • Comparison operators
  • Logical operators
  • Bitwise operators
  • Assignment Operator
  • Identity operators
  • Membership operators

1) Arithmetic Operators

These Python arithmetic operators include Python operators for basic mathematical operations.

Arithmetic Operators

Operator TypeDefinition
+Addition operator
-Subtraction operator
*Multiplication operator
/Division operator
%Reminder operator
//Divides and returns the value of the remainder.
**Raises the left operand to the power of right.
x = 15
y = 10

print('x + y =', x+y)
print('x - y =', x-y)
print('x * y =', x*y)
print('x / y =', x/y)
print('x % y =', x%y)
print('x // y =', x//y)
print('x ** y =', x**y)

>>> x + y = 25
>>> x – y = 5
>>> x * y = 150
>>> x / y = 1.5
>>> x % y = 5
>>> x // y = 1
>>> x ** y = 576650390625

2) Comparison Operators

Comparison operators perform a comparison between two values. It returns a boolean True or False depending upon the result of the comparison.

Comparison Operators

OperatorsDefinition
>True if left operand is greater than the right
<True if left operand is less than the right
==True if both operands are equal
!=True if operands are not equal
>=True if left operand is greater than or equal to the right
<=True if left operand is less than or equal to the right
x = 8
y = 15

print('x>y is', x>y)
print('x< y is',  x<y)
print('x == y is', x==y)
print('x != y is', x!=y)
print('x >= y is', x>=y)
print('x<= y is', x<=y)

>>> x > y is False
>>> x < y is True
>>> x == y is False
>>> x != y is True
>>> x >= y is False
>>> x <= y is True

3) Logical Operators

Logical operators are used for performing AND, OR and NOT operations. It either returns True or False according to the condition.

Logical Operators

OperatorsDefinitions
andTrue if both the operands are true
orTrue if either of the operands is true
notTrue if operand is false
a = True
b = False

print(‘a and b is’, a and b)
print(‘a or b is’, a or b)
print(‘not a is’, not a)

>>> a and b is False
>>> a or b is True
>>> not a is False

6) Identity Operators

Python offers 2 types of identity operators i.e is and is not.

Both are used to compare if two values are located on the same part of the memory. Two variables that are equal does not imply that they are identical.

Identity Operators

OperatorsDefinitions
isTrue if the operands are identical
is notTrue if the operands are not identical

7) Membership Operators

Python offers 2 types of membership operators i.e in and not in.

Both are used to test whether a value or variable is in a sequence.
